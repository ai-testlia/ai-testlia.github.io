中二銜接練習：人物情緒表達
（校園與生活篇）

潘SIR中文教室編撰

💡 學習目標：本練習專為協助同學跳脫「英式中文」思維而設。透過豐富的校園與生活情境，幫助同學掌握精準的情緒詞彙及成語，提升日常寫作的自然流暢度。

一、供詞填充

請從下表中選出最適合的詞語填在橫線上（每個詞語只可使用一次）。

忐忑不安
欣喜若狂
愁眉苦臉
勃然大怒
垂頭喪氣
驚惶失措
悶悶不樂
喜出望外
心急如焚
啼笑皆非
1.知道明天要派發期終考試的成績表，我整晚都感到，在床上翻來覆去，無法入睡。
✅ 答案：忐忑不安
2.當發現自己辛苦儲錢買的遊戲機被弟弟弄壞了，哥哥頓時，大聲責罵他。
✅ 答案：勃然大怒
3.聽到媽媽答應暑假帶我們全家去日本旅行，妹妹高興得，在客廳裡跳來跳去。
✅ 答案：欣喜若狂
4.測驗不及格的明華地走回座位，把頭埋在雙臂間，一句話也不說。
✅ 答案：垂頭喪氣
5.在商場裏走失了的小男孩地四處張望，找不到媽媽的他忍不住大哭起來。
✅ 答案：驚惶失措
6.因為跟最好的朋友吵架了，家明這幾天都，連平時最愛的籃球也提不起勁打。
✅ 答案：悶悶不樂
7.原本以為今年生日沒有禮物，沒想到爸爸偷偷買了我最想要的球鞋，真讓我感到。
✅ 答案：喜出望外
8.看着年幼的妹妹把積木當成食物塞進玩具熊的嘴裏，媽媽真是感到。
✅ 答案：啼笑皆非
9.班長不小心遺失了全班同學的秋季旅行通告，嚇得，急得像熱鍋上的螞蟻。
✅ 答案：心急如焚
10.為什麼你今天一整天都的？是不是在學校遇到了什麼煩心事？
✅ 答案：愁眉苦臉
二、不供詞填充

請根據句子意思及括號內的提示，填寫最合適的情緒詞語（須為兩個字的詞語）。

1.經過三個月的刻苦訓練，校隊終於奪得校際籃球比賽的冠軍，大家在頒獎台上流下了的眼淚。[提示：因高興或受到刺激而感情強烈]
✅ 答案：激動
2.我不小心把同學借給我的精美筆記本弄不見了，現在心裏感到非常，不知如何向他交代。[提示：做錯事後感到後悔及自責]
✅ 答案：懊悔
3.明明是同桌同學說話，但老師卻誤會是我，並當眾責備我。我感到十分，眼淚一直在眼眶裡打轉。[提示：受到不公平對待而心裏難受]
✅ 答案：委屈
4.每次做專題研習遇到困難，他總是先組員沒有盡力，而不是自己想辦法解決問題。[提示：心中不滿而責怪別人]
✅ 答案：抱怨
5.看到好朋友結交了新同伴而忽略自己，他的心裏不禁產生了一絲。[提示：因別人比自己好或搶走關注而心生怨恨]
✅ 答案：嫉妒
6.晚上自己一個人在客廳看完了那套恐怖電影後，我對黑暗產生了莫名的，連去洗手間也要開燈。[提示：感到害怕、驚恐]
✅ 答案：恐懼
7.考試終於完結了，我們全班同學都熱切地着下星期的畢業旅行。[提示：充滿期待地盼望]
✅ 答案：期盼
8.剛轉學到這裏，我還沒有交到新朋友，每次小息一個人在操場漫步，總覺得有點。[提示：單身無靠，感到寂寞]
✅ 答案：孤單
9.平時數學成績普通的他，這次竟然考了全班第一，大家都對這個突如其來的進步感到很。[提示：覺得奇怪、驚奇]
✅ 答案：詫異
10.弟弟一直在旁邊大吵大鬧，不斷打斷我溫習英文，讓我感到十分。[提示：因為被打擾而嫌煩、討厭]
✅ 答案：厭煩
三、改寫句子

以下句子的語氣過於口語化，或受了「英式中文（English translated logic）」的影響。請以意思不變為原則，把它們改寫成流暢自然的標準書面語。

1.我很嬲，因為他 break 了我的 promise。
✅ 參考答案：我很生氣，因為他對我食言了。（或：他沒有遵守諾言，讓我感到十分生氣。）
2.我感到好 down，因為我不 pass 個 test。
✅ 參考答案：測驗不及格，讓我感到十分沮喪。（或：因為測驗不及格，我感到很失落。）
3.我很驚當我看見那隻流浪狗。
✅ 參考答案：看見那隻流浪狗時，我感到很害怕。
4.這是一件讓我變得很傷心的一件事。
✅ 參考答案：這件事讓我感到十分傷心。
5.我有很多 stresses 在我的學習上。
✅ 參考答案：在學習上，我承受了很大的壓力。
6.我很 happy 去看電影和我的朋友一起。
✅ 參考答案：能和朋友一起看電影，我感到很高興。
7.老師大聲話我，我覺得好冇面。
✅ 參考答案：被老師大聲責罵，我感到十分羞愧。（或：感到十分難堪）
8.我真的很不明白 why 你要這樣對我，我覺得好心痛。
✅ 參考答案：我實在不明白你為何要這樣對待我，這讓我感到很心痛。
9.我 feel 到好 boring 當我聽這個 talk。
✅ 參考答案：聽這個講座時，我感到十分沉悶。
10.我的媽媽非常之 proud of 我。
✅ 參考答案：我的媽媽為我感到十分自豪。
四、重組句子

請將下列散落的詞組重新組合成邏輯嚴謹、文句通順的書面語句子（需加上適當的標點符號）。

1.感到 / 十分 / 的 / 他 / 為 / 粗心大意 / 懊惱 / 自己
✅ 參考答案：他為自己的粗心大意感到十分懊惱。
2.滿懷 / 步出 / 考試 / 信心 / 地 / 禮堂 / 同學們
✅ 參考答案：同學們滿懷信心地步出考試禮堂。
3.弟弟 / 讓 / 感到 / 做法 / 莫名其妙 / 的 / 我
✅ 參考答案：弟弟的做法讓我感到莫名其妙。
4.嚇得 / 面色 / 突然 / 蒼白 / 傳來的 / 巨響 / 妹妹
✅ 參考答案：突然傳來的巨響嚇得妹妹面色蒼白。
5.終於 / 鬆了一口氣 / 尋回 / 遺失的 / 手機 / 哥哥 / 後
✅ 參考答案：哥哥尋回遺失的手機後，終於鬆了一口氣。
6.氣得 / 滿面 / 通紅 / 他的 / 謊言 / 媽媽 / 說
✅ 參考答案：他說的謊言氣得媽媽滿面通紅。
7.因為 / 感到 / 班長 / 老師的 / 十分 / 稱讚 / 鼓舞 / 受到
✅ 參考答案：班長因為受到老師的稱讚而感到十分鼓舞。
8.忍不住 / 委屈的 / 起來 / 淚水 / 小女孩 / 流下 / 痛哭
✅ 參考答案：小女孩忍不住流下委屈的淚水，痛哭起來。
9.焦急 / 徘徊 / 爸爸 / 地 / 訓導處外 / 在
✅ 參考答案：爸爸在訓導處外焦急地徘徊。
10.總是 / 樂觀的 / 感染 / 笑容 / 身邊的 / 能 / 朋友 / 她
✅ 參考答案：她樂觀的笑容總是能感染身邊的朋友。
五、段落理解

閱讀以下兩段短文，並回答問題。

【文章一】
今天是派發默書成績的日子。我握着那張只有四十分的測驗紙，心跳得極快，雙手微微發抖。回想起昨晚，我為了解鎖電子遊戲的新關卡而沒有溫習，現在只有滿滿的懊悔。放學後，我拖着沉重的步伐走回家，腦海中不斷浮現媽媽生氣的面容。當我膽怯地把測驗紙遞給媽媽時，她竟然沒有破口大罵，只是嘆了一口氣，眼神裏充滿了失望。那一刻，我感到無比羞愧，暗暗下定決心下次一定要努力。
1.派發成績時，「我」的身體出現了甚麼反映緊張情緒的生理反應？
✅ 參考答案：他的心跳得極快，雙手微微發抖。
2.文中哪個詞語準確地概括了「我」對昨晚貪玩行為的感受？
✅ 參考答案：懊悔。
3.為甚麼「我」放學時會「拖着沉重的步伐」？
✅ 參考答案：因為他的成績不理想，心裡害怕面對媽媽的責備，心情十分沉重。
4.媽媽看到成績後，她的情緒是怎樣的？
✅ 參考答案：媽媽感到十分失望。
5.看到媽媽的反應後，作者最後的情緒出現了甚麼轉變？
✅ 參考答案：他感到無比羞愧，並下定決心下次要努力溫習。

【文章二】
小明和小強是形影不離的好朋友。星期五的小息，小強不小心把墨水倒在小明新買的筆袋上。小明頓時火冒三丈，大聲責怪小強。小強感到十分委屈，連忙道歉，但小明氣得轉身就走。整個週末，小明都覺得心裏悶悶的，他看着那個筆袋，其實只是一小塊污漬。他開始後悔自己當時太衝動，明白到友情比一個筆袋珍貴得多。星期一早上，小明主動向小強說對不起，兩人相視而笑，冰釋前嫌。
6.小明新買的筆袋被弄髒後，他第一時間有甚麼情緒反應？
✅ 參考答案：他頓時火冒三丈（非常生氣）。
7.試寫出文中表示小強被責罵後難受心情的詞語。
✅ 參考答案：委屈。
8.週末時，小明的心情是怎樣的？
✅ 參考答案：他覺得心裏悶悶的，並且為自己的衝動感到後悔。
9.讓小明心情從「生氣」轉變為「後悔」的原因是什麼？
✅ 參考答案：因為他明白到友情比一個筆袋珍貴得多。
10.故事結局中，「兩人相視而笑」代表他們的情緒變得怎樣？
✅ 參考答案：代表他們釋懷了（不再生氣），和好如初，心情回復開心。
