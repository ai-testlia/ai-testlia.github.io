第一部分：核心詞彙精解與應用
高頻實用語詞教學

在寫作或閱讀理解中，準確運用詞語是取分的關鍵。以下是校園及生活題材中常見的「高級詞彙」。

落實
將計劃、政策或決定付諸實行。
✅ 搭配：落實方案 / 落實分工
優化
加以改善，使事物變得更加完善、優秀。
✅ 搭配：優化系統 / 優化溝通
協調
使各方面配合得當，步調一致。
✅ 搭配：協調工作 / 協調各界
機制
泛指一個組織或系統的運作規則與流程。
✅ 搭配：投訴機制 / 獎勵機制
評估
對人或事物的價值、風險或影響進行評價。
✅ 搭配：評估風險 / 評估性能
侷限
限制在某個狹窄的範圍內。
✅ 搭配：受預算侷限 / 打破侷限
範疇
領域、範圍（多指學術、工作或活動）。
✅ 搭配：學習範疇 / 服務範疇
趨勢
事物發展的動向或方向。
✅ 搭配：發展趨勢 / 社會趨勢

第二部分：必學成語與語境配對
校園與生活實用成語
成語 釋義精解 寫作語境示範
言簡意賅 言語簡練，但意思完備無遺。 上台演講時，與其長篇大論，不如言簡意賅地帶出重點。
墨守成規 思想保守，守著老規矩不肯改變。 時代不斷進步，我們不能墨守成規，必須勇於創新。
如履薄冰 像走在薄冰上，比喻行事極為謹慎。 處理這份極機密的文件，他如履薄冰，不敢有半點馬虎。
未雨綢繆 在未下雨前就先修補好門窗，比喻事先做好準備。 出發旅行前，媽媽總會未雨綢繆，把各類平安藥準備妥當。
截然不同 形容事物或情況完全不同。 這對雙胞胎的外貌雖然相似，但性格卻截然不同。
面面俱到 各方面都照顧到，沒有遺漏。 班長處事大公無私，總能做到面面俱到，深得同學信賴。
開門見山 比喻說話或寫文章直截了當，一開始就進入正題。 既然時間緊迫，我就開門見山地說出這次會議的目的吧。
事半功倍 花費一半的力氣，卻能得到雙倍的效果。 只要掌握正確的溫習技巧，學習自然能事半功倍。
💡
考試技巧
填充成語時，必須留意句子的「前文後理」。例如看到「不肯改變」、「老規矩」，即聯想「墨守成規」；看到「直截了當」、「不兜圈子」，即聯想「開門見山」。

第三部分：句式結構與改寫技巧
遠離「口語化」與「歐化中文」

許多同學在寫作時，經常會不自覺地混入廣東話口語，或者受到英文文法影響（即歐化中文），導致文句累贅、不通順。

📝 常見歐化語病診斷
❌ 濫用「作出」：作出調查 ➔ ✅ 調查
❌ 濫用「被」字：被他發現 ➔ ✅ 遭他發現 / 讓他發現了
❌ 直譯名詞作動詞：給出一個建議 (make a suggestion) ➔ ✅ 建議
❌ 英式邏輯 / 口語化
如果你唔快啲交功課，老師會變到好生氣。 (You will make the teacher become angry)
✅ 自然書面語
如果你不盡快提交功課，老師會非常生氣。
關聯詞與句構重組

在重組句子中，找出「關聯詞」是成功的一半。常見結構包括：

選擇 / 比較結構
與其（互相抱怨浪費時間），不如（抓緊時間溫習）。
條件結構
只有（努力練習），我們才能（在比賽中獲勝）。
📝 隨堂練習：改寫句子（轉化為標準書面語）

第四部分：閱讀與段落理解技巧
閱讀理解拆題法

應付閱讀理解，首先要學會從段落中尋找線索。以下是兩個非常實用的技巧：

技巧一：留意「總結性」字眼
段落中若出現「總括而言」、「因此」、「由此可見」，其後多為段落的中心主旨。回答「段落大意」或「作者想分享的道理」時，需從這些句子中提煉答案。
技巧二：利用「前文後理」推敲詞義
若題目要求解釋詞語（例如：優化、範疇），不要死背字典解釋。嘗試把詞語從句子中抽離，看看用甚麼日常詞彙替換後，句意依然通順，這便是最佳答案。
📝 隨堂練習：段落理解
【段落一】
「在學校生活中，與同學相處是一門大學問。與其遇到誤會時互相猜忌，不如開門見山地把問題說清楚。同時，我們應該隨時優化自己的溝通方式，力求言簡意賅，因為這樣不但能減少不必要的爭吵，更能和大家建立深厚的友誼。」
1. 文中建議遇到誤會時應該怎麼做？
👁️ 觀看答案
✏️ 答案：開門見山地把問題說清楚。
2. 為什麼與同學說話時要力求「言簡意賅」？
👁️ 觀看答案
✏️ 答案：因為這樣能減少不必要的爭吵，與大家建立深厚友誼。
3. 「優化」在文中是指什麼？
👁️ 觀看答案
✏️ 答案：改善 / 改進（說話或溝通的方法）。
4. 這段文字主要想分享什麼道理？
👁️ 觀看答案
✏️ 答案：坦誠與精煉的溝通對與同學相處非常重要。
5. 請為這段文字加一個合適的小標題。
👁️ 觀看答案
✏️ 答案：同學之間的相處之道 / 溝通的藝術（意近即可）。
【段落二】
「網絡遊戲近年已成為青少年的主要社交媒介，甚至成為一種趨勢。雖然打機在放鬆心情和廣結朋友等範疇都有其好處，但同學必須注意時間的侷限，不能玩物喪志。因此，我們在享受打機樂趣的同時，也必須認真評估它對我們日常生活和學業帶來的長遠影響。」
6. 文中提到的「趨勢」是指什麼？
👁️ 觀看答案
✏️ 答案：網絡遊戲成為青少年的主要社交媒介。
7. 根據段落，打機有哪些好處？（列出兩項）
👁️ 觀看答案
✏️ 答案：放鬆心情、廣結朋友。
8. 作者提醒同學打機時要注意什麼限制？
👁️ 觀看答案
✏️ 答案：時間的限制（不能玩物喪志）。
9. 文中「範疇」一詞可以換成哪一個日常詞語？
👁️ 觀看答案
✏️ 答案：領域 / 方面。
10. 為什麼我們在打機的同時，還需要「評估」其長遠影響？
👁️ 觀看答案
✏️ 答案：因為要確保打機不會對日常生活和學業造成負面影響。
