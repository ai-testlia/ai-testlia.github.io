練習定位
本卷屬第三型進階練習（專業對弈），目標由「通順」進入「典雅」與「精準」。詞彙涉及決策、管理及宏觀分析術語；成語側重策略思考與職場博弈。

PART THREE
改寫句子
每題 2 分・共 20 分
⚠️
常見陷阱
改寫後須呈現「行政色彩」的書面語：避免口語化語氣詞（如「吓」、「好正」）、英語借詞（如 Plan、Report、Show），改以正式詞彙替代。
改寫技巧要點
識別口語／英語詞
→
換成書面詞彙
→
調整句式結構
→
確認語境正式
第 1 題 【口語】
2分
你最好先同我講吓你個 Plan 係點。
✅ 參考答案：煩請閣下先向本人講解貴方案的具體內容。
第 2 題 【英式】
2分
如果我哋唔快啲做嘢，我哋會輸好多錢。
✅ 參考答案：倘若我方未能及時採取行動，恐將蒙受重大的財務損失。
第 3 題 【口語】
2分
佢係一個好老細，所有人都好鍾意佢。
✅ 參考答案：他是一位深受同仁愛戴的出色管理者。
第 4 題 【英式】
2分
我寫呢份 Report 係想 Show 俾你睇個 Project 嘅進度。
✅ 參考答案：本報告旨在向閣下呈示本項目目前的推進進度。
第 5 題 【口語】
2分
呢個方法雖然好似好正，但其實好難做到。
✅ 參考答案：此方案表面上固然理想，然而實際執行上存在相當大的困難。
第 6 題 【英式】
2分
非常多謝你嘅 Suggestion，我哋會考慮吓。
✅ 參考答案：衷心感謝閣下提出寶貴意見，我方將認真研究並作出考量。
第 7 題 【口語】
2分
你咁樣做會整壞公司個名，你要諗清楚。
✅ 參考答案：閣下此舉恐將損害公司的聲譽，懇請三思而後行。
第 8 題 【英式】
2分
我哋一定要 Meet 個 Deadline，唔係就大鑊。
✅ 參考答案：我方務必於截止期限前完成任務，否則後果將不堪設想。
第 9 題 【口語】
2分
佢講嘢好玄，搞到大家都有啲 Confuse。
✅ 參考答案：其發言措辭晦澀，令在座各人未能充分理解其意思。
第 10 題 【口語】
2分
我哋要搵出個問題嘅根源，唔好只係執手尾。
✅ 參考答案：我方應深入探究問題的根本成因，而非僅作表面的補救處理。

PART FOUR
重組句子
每題 2 分・共 20 分
💡
答題策略
先找主語，再定謂語，最後安排修飾語位置。留意關聯詞（倘若、既…又、雖然…卻）能提示句子結構。
第 1 題
2分
詞組：倘若 / 恐怕 / 難以 / 方案 / 落實 / 達成 / 預期 / 目標 / 準確
✅ 倘若方案未能準確落實，恐怕難以達成預期目標。
第 2 題
2分
詞組：旨在 / 這種 / 資源 / 分配 / 效益 / 最大化 / 讓 / 核心 / 競爭力 / 並 / 提升
✅ 這種資源分配旨在讓效益最大化，並提升核心競爭力。
第 3 題
2分
詞組：既 / 他的 / 方案 / 具備 / 宏觀 / 視野 / 又 / 實踐 / 兼顧 / 細節 / 的
✅ 他的方案既具備宏觀視野，又兼顧實踐的細節。
第 4 題
2分
詞組：有見及此 / 決定 / 調整 / 策略 / 管理層 / 以 / 應對 / 挑戰 / 市場 / 的
✅ 有見及此，管理層決定調整策略，以應對市場的挑戰。
第 5 題
2分
詞組：雖然 / 艱巨 / 任務 / 我們 / 卻 / 從未 / 動搖 / 目標 / 達成 / 的 / 決心
✅ 雖然任務艱巨，我們卻從未動搖達成目標的決心。
第 6 題
2分
詞組：洞察 / 變化 / 領導者 / 能夠 / 敏銳地 / 市場 / 並 / 契機 / 尋找 / 發展
✅ 領導者能夠敏銳地洞察市場變化，並尋找發展契機。
第 7 題
2分
詞組：由於 / 因素 / 計劃 / 被迫 / 技術 / 延期 / 實施 / 導致 / 損失 / 額外
✅ 由於技術因素，計劃被迫延期實施，導致額外損失。
第 8 題
2分
詞組：與其 / 互相推諉 / 不如 / 共同 / 責任 / 承擔 / 尋找 / 出路
✅ 與其互相推諉，不如共同承擔責任，尋找出路。

✅ 為了規範內部流程，決定建立完善的監管機制。

PART FIVE
段落理解
每題 2 分・共 20 分
【段落一】
「在企業管理中，審慎的財務決策與大膽的技術變革同等重要。管理者若只專注於短期效益，往往會忽視長遠的風險。因此，建立一套透明且高效的溝通機制，不但能促進部門間的磨合，更能確保企業在動盪的市場中保持核心競爭力。」
段落一：理解題
第 1 題
2分
管理者如果只追求短期效益，會有什麼後果？
✅ 往往會忽視長遠的風險，危及企業的可持續發展。
第 2 題
2分
文中提到哪兩樣東西在管理中「同等重要」？
✅ 「審慎的財務決策」與「大膽的技術變革」。
第 3 題
2分
建立溝通機制有哪些具體好處？（列出兩項）
✅ ①促進部門間的磨合；②確保企業在動盪市場中保持核心競爭力。
第 4 題
2分
請解釋文中「審慎」在財務決策上的重要性。
✅ 財務決策一旦失誤，後果重大且難以逆轉。「審慎」意味著在決策前充分評估風險，避免因草率行事而蒙受財務損失，確保企業財務穩健。
第 5 題
2分
這段文字主要想表達什麼管理智慧？
✅ 優秀的管理者應兼顧穩健與創新，在審慎決策的同時敢於變革；並透過建立溝通機制，令企業在逆境中維持競爭力。
【段落二】
「職場中的情緒智商（EQ）往往比技術能力更具關鍵影響。一位能隨機應變且懂得面面俱到的領袖，更能贏得下屬的推心置腹。面對利益衝突時，若能以互惠互利為原則，往往能化解矛盾，甚至將危機轉化為合作的契機。」
段落二：理解題
第 6 題
2分
文中認為在職場中，什麼能力比技術能力更重要？
✅ 情緒智商（EQ）。
第 7 題
2分
一位成功的領袖應該具備哪兩種特質？（運用文中四字詞）
✅ ①隨機應變；②面面俱到。
第 8 題
2分
面對衝突時，應以什麼原則來處理？
✅ 以「互惠互利」為原則。
第 9 題
2分
「推心置腹」在文中形容什麼樣的關係？
✅ 形容下屬對領袖的信任，願意真誠地向其傾訴心聲，即一種坦誠相待、互相信賴的上下屬關係。
第 10 題
2分
根據段落，危機在什麼情況下可以變成「契機」？
✅ 當各方能以「互惠互利」為原則處理利益衝突，化解矛盾時，危機便可轉化為雙方合作的契機。
