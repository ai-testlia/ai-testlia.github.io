語文基礎強化筆記

中文能力強化練習
主題：學習環境的情緒表達
（測驗與專題研習篇）

潘SIR中文教室編撰

💡 學習目標：不少同學寫情緒感受時，仍不自覺地用「英式中文」的直譯思維遣詞。這份練習從你熟悉的校園生活、溫習和測考場景出發，帶你掌握地道的情緒詞彙和成語，讓日常寫作讀起來自然順暢。

一、供詞填充

📖 詞語教學：在完成下面的填充練習前，請先掌握以下十個詞語的意思和用法。

1. 焦頭爛額
解釋：事情多到應付不來，忙得狼狽不堪。
例句：面對突如其來的大量功課，他忙得焦頭爛額。
2. 恍然大悟
解釋：原本想不通，忽然一下子完全明白過來。
例句：聽老師講解後，他終於恍然大悟，明白了解題的關鍵。
3. 絞盡腦汁
解釋：想破了頭，用盡一切辦法去思考。
例句：他絞盡腦汁，終於想出一個有趣的故事開頭。
4. 如釋重負
解釋：放下心頭大石後，頓時感到輕鬆。
例句：考試結束後，他終於如釋重負地舒了一口氣。
5. 心灰意冷
解釋：失去信心和鬥志，情緒低落消沉。
例句：一再落選後，他一度心灰意冷，不想再參加比賽。
6. 忐忑不安
解釋：心裡七上八下，擔心又不安。
例句：等待成績公佈時，她感到忐忑不安。
7. 沾沾自喜
解釋：自覺得意，還忍不住表現出來。
例句：他因為考了滿分而沾沾自喜，四處向人炫耀。
8. 胸有成竹
解釋：事前準備充足，做起事來從容不迫。
例句：經過反覆練習，他對這次演講已胸有成竹。
9. 興高采烈
解釋：情緒高漲，開心得不得了。
例句：得知學校旅行的消息後，同學們都興高采烈。
10. 愁眉苦臉
解釋：滿臉愁容，一副心事重重的樣子。
例句：他因為默書不及格而愁眉苦臉。

請從下表中選出最適合的四字詞語填在橫線上（每個詞語只可使用一次）。

1.距離呈分試只剩下一星期，面對堆積如山的溫習資料，大衛感到。
✅ 答案：焦頭爛額
2.經過老師悉心講解，原本對這道科學難題一頭霧水的我終於。
✅ 答案：恍然大悟
3.為了想出一條引人入勝的演講題目，他整晚，連晚飯也忘了吃。
✅ 答案：絞盡腦汁
4.交上那份耗時一個月才完成的歷史專題報告後，組員們都感到。
✅ 答案：如釋重負
5.雖然這次測驗成績退步了，但你也不必，只要找出弱點，下次一定能考好。
✅ 答案：心灰意冷
6.明天就是中文口試，平時很少說中文的約翰感到，手心直冒汗。
✅ 答案：忐忑不安
7.看到測驗卷上滿分的紅字，志明不禁，到處向同學炫耀。
✅ 答案：沾沾自喜
8.溫習準備得十分充足的班長，對這次期末考試，毫不緊張。
✅ 答案：胸有成竹
9.知道學校下星期舉辦電競比賽後，熱愛打機的男同學們都地討論戰術。
✅ 答案：興高采烈
10.由於連續兩次默書都不及格，家明整天，擔心會被家長責罵。
✅ 答案：愁眉苦臉
二、不供詞填充

📖 詞語教學：在完成下面的填充練習前，請先掌握以下十個詞語的意思和用法。

1. 困惑
解釋：心裡感到疑惑，不知道該怎麼辦或不明白事情的道理。
例句：面對艱深的文言文，他讀得十分困惑。
2. 氣餒
解釋：遇到挫折後失去信心和勇氣，情緒低落。
例句：雖然初賽落選，但她沒有氣餒，決心再接再厲。
3. 專注（同「專心」）
解釋：把注意力集中在一件事上，不受外界干擾。
例句：他專注地做功課，完全沒聽見媽媽的呼喚。
4. 煩躁
解釋：心情煩悶焦急，容易發脾氣。
例句：塞車令他等得十分煩躁。
5. 焦急
解釋：因擔心某事而心情急切不安。
例句：媽媽遲遲未到，小明焦急地張望着。
6. 驚喜
解釋：出乎意料的驚訝和喜悅。
例句：生日那天，朋友的突然到訪讓她十分驚喜。
7. 期待（同「盼望」）
解釋：心裡懷着希望，等候某事發生。
例句：他十分期待這次的畢業旅行。
8. 輕鬆
解釋：沒有壓力或負擔，感覺舒暢自在。
例句：考試結束後，他頓時感到一身輕鬆。
9. 畏懼（同「害怕」）
解釋：心裡感到恐懼，不敢面對。
例句：他對於在眾人面前發言感到十分畏懼。
10. 懊悔（同「後悔」）
解釋：對已經做錯的事情感到自責、悔恨。
例句：他因臨考前貪玩而沒溫習，結果不及格，感到十分懊悔。

請根據句子意思及括號內的提示，填寫最合適的情緒詞語（須為兩個字的詞語）。

1.面對這篇充滿生字和文言文的閱讀理解，我感到十分，不知從何入手。[提示：感到疑難，不知所措]
✅ 答案：困惑
2.努力溫習了兩星期，但測驗依然不及格，這讓他感到十分。[提示：失去信心和勇氣]
✅ 答案：氣餒
3.只要戴上耳機，聽着純音樂，我就能地完成繁重的功課。[提示：集中精神，全神貫注]
✅ 答案：專注 / 專心
4.哥哥正在努力準備明天的測驗，但窗外不斷傳來地盤施工的噪音，讓他感到很。[提示：心情煩悶急躁]
✅ 答案：煩躁
5.距離網上遞交功課的限期只剩下五分鐘，但電腦突然死機了，嚇得我十分。[提示：着急、焦慮]
✅ 答案：焦急
6.本以為這次默書一定會不及格，沒想到竟然剛好及格了，真是讓我感到。[提示：又驚又喜]
✅ 答案：驚喜
7.終於考完試了，同學們都非常即將到來的聖誕節聯歡會。[提示：充滿期待地盼望]
✅ 答案：期待 / 盼望
8.考完最後一科，走出禮堂的那一刻，我的心情頓時變得無比。[提示：不感到有負擔或壓力]
✅ 答案：輕鬆
9.每次被老師抽中要在全班面前用中文演講，不擅長表達的瑪麗都會感到。[提示：害怕、畏縮]
✅ 答案：畏懼 / 害怕
10.如果我昨晚沒有沉迷打機，今天就不會在課堂上打瞌睡了，現在回想起來真是。[提示：做錯事或說錯話後感到後悔]
✅ 答案：懊悔 / 後悔
三、改寫句子

以下句子的語氣過於口語化，或受了「英式中文（English translated logic）」的影響。請以意思不變為原則，把它們改寫成流暢自然的標準書面語。

1.我感到很大 stress 當我在做中文功課的時候。
✅ 參考答案：做中文功課時，我感到壓力很大。（或：做中文功課讓我感到很大的壓力。）
2.我的腦一片空白當我看見那條好難的數學題。
✅ 參考答案：看到那道艱深的數學題時，我的腦海一片空白。
3.這個 group project 讓我變得很頭痛。
✅ 參考答案：這份小組專題研習讓我感到十分頭痛。
4.我覺得很 confused 對於老師剛才講的 grammar rules。
✅ 參考答案：我對老師剛才講解的語法規則感到十分困惑。
5.我很 worry 我的 report card 會不靚。
✅ 參考答案：我很擔心自己的成績單不理想。
6.他很 proud 當他拿到全班第一名。
✅ 參考答案：考取全班第一名後，他感到十分自豪。
7.我 feel 到好 relief 當我 submit 了那篇 essay。
✅ 參考答案：遞交了那篇作文後，我頓時感到如釋重負。
8.為什麼你 look 到好 sad？你 fail 了這個 test 嗎？
✅ 參考答案：為什麼你看起來悶悶不樂？是測驗不及格嗎？
9.我今天沒有 mood 去溫習 dictation。
✅ 參考答案：我今天沒有心情溫習默書。
10.這個老師的堂很 boring，我快要睡着了。
✅ 參考答案：這位老師的課很沉悶，我快要睡着了。
四、重組句子

請將下列散落的詞組重新組合成邏輯嚴謹、文句通順的書面語句子（需加上適當的標點符號）。

1.面對 / 難題 / 他 / 感到 / 複雜的 / 數學 / 十分 / 困惑
✅ 參考答案：面對複雜的數學難題，他感到十分困惑。
2.終於 / 專題研習 / 完成了 / 我們 / 感到 / 如釋重負 / 都
✅ 參考答案：終於完成了專題研習，我們都感到如釋重負。
3.讓 / 突如其來的 / 感到 / 測驗 / 同學們 / 驚惶失措
✅ 參考答案：突如其來的測驗讓同學們感到驚惶失措。
4.受到 / 他 / 老師的 / 感到 / 稱讚 / 沾沾自喜 / 後 / 十分
✅ 參考答案：他受到老師的稱讚後，感到十分沾沾自喜。
5.專心地 / 同學們 / 都在 / 溫習 / 圖書館裏 / 準備 / 考試
✅ 參考答案：同學們都在圖書館裏專心地溫習準備考試。
6.絞盡腦汁 / 他 / 構思 / 為了 / 題目 / 寫作 / 整晚 / 都在
✅ 參考答案：為了構思寫作題目，他整晚都在絞盡腦汁。
7.成績 / 面對 / 的 / 感到 / 退步 / 心灰意冷 / 她 / 十分
✅ 參考答案：面對退步的成績，她感到十分心灰意冷。
8.期待着 / 充滿 / 同學們 / 畢業旅行 / 的 / 喜悅地 / 到來
✅ 參考答案：同學們充滿喜悅地期待着畢業旅行的到來。
9.滿懷 / 步出 / 考試 / 信心 / 地 / 禮堂 / 他
✅ 參考答案：他滿懷信心地步出考試禮堂。
10.聽完 / 恍然大悟 / 老師的 / 終於 / 講解後 / 我 / 明白了
✅ 參考答案：聽完老師的講解後，我恍然大悟，終於明白了。
五、段落理解

閱讀以下兩段短文，並回答問題。

【文章一】
今個星期的電腦課，老師要求我們分組編寫一個小遊戲程式。一開始，我對編程語法一竅不通，看着螢幕上密密麻麻的代碼，感到無比困惑和煩躁。我嘗試修改了幾次錯誤，但程式依然無法運行，那種挫敗感讓我幾乎想放棄。幸好，同組的彼得耐心地為我逐行解釋邏輯。在他的指導下，我突然恍然大悟，找到了程式碼裏的漏洞。當看到小遊戲終於成功在螢幕上運行時，我們兩人都興奮得擊掌歡呼，之前的焦慮和氣餒瞬間煙消雲散。
1.看到螢幕上密密麻麻的代碼時，作者最初的情緒是怎樣的？
✅ 參考答案：他感到無比困惑和煩躁。
2.為甚麼作者會產生挫敗感，甚至萌生放棄的念頭？
✅ 參考答案：因為他嘗試修改了幾次錯誤，但程式依然無法運行。
3.文中哪個四字成語形容作者得到彼得指導後，突然明白過來的感覺？
✅ 參考答案：恍然大悟。
4.當遊戲成功運行時，他們的甚麼肢體動作反映了高興激動的情緒？
✅ 參考答案：他們興奮得擊掌歡呼。
5.作者在這個編程過程中，情緒經歷了怎樣的轉變？
✅ 參考答案：他的情緒由最初的困惑、煩躁及氣餒，最終轉變為興奮高興。

【文章二】
隨着期末考試的腳步越來越近，班上的氣氛變得異常緊張。平時愛開玩笑的明華，這幾天也變得沉默寡言，眉頭深鎖。昨天放學後，我看到他獨自留在課室裏溫習數學，不時抓着頭髮，發出無奈的嘆息。我走過去拍拍他的肩膀，他抬起頭，眼神中流露出一絲擔憂。他告訴我，如果這次數學再不及格，就會被父母禁止參加暑假的籃球訓練營。為了幫助他減輕壓力，我提議每天小息陪他一起操練歷屆試題，希望能讓他重拾信心，不再那麼忐忑不安。
6.臨近期末考試，班上的氣氛有何變化？
✅ 參考答案：班上的氣氛變得異常緊張。
7.作者透過甚麼外在行為，察覺到明華感到煩惱？
✅ 參考答案：明華變得沉默寡言、眉頭深鎖，並不時抓着頭髮和發出嘆息。
8.明華感到擔憂的真正原因是什麼？
✅ 參考答案：他害怕數學不及格，會被父母禁止參加暑假籃球訓練營。
9.文中哪個四字成語形容明華心中擔憂，感到不安的狀態？
✅ 參考答案：忐忑不安。
10.作者如何幫助明華改善負面情緒？
✅ 參考答案：作者提議每天小息陪他一起操練歷屆試題，幫助他重拾信心及減輕壓力。
