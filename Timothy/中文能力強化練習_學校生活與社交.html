潘SIR的話
這份練習專為習慣英語思考的同學而設。我們在開始練習前，先溫習一下日常校園和社交中最常用到的詞彙和成語。做練習時，試著把腦海中的「英文翻譯」關掉，用語感去感受句子的連貫性！（完成題目後可點擊「觀看答案」按鈕核對）
📖

一、本課詞語及成語詳解

📌 核心詞彙（供詞填充用語）
詞語 釋義 生活化例句
踴躍
 形容情緒熱烈，積極參與。（Eager / Enthusiastic）
老師一宣佈舉辦電競比賽，同學們就踴躍報名參加。
融洽
 形容彼此感情很好，沒有矛盾。（Harmonious）
我們組員之間的關係十分融洽，從來不會因為分工而吵架。
尷尬
 處境困難，令人感到難為情或不知所措。（Awkward）
我在走廊不小心叫錯了別班同學的名字，當時真的非常尷尬。
默契
 雙方不需要語言表達就能明白對方的心意。（Chemistry）
他們兩個在籃球場上充滿默契，一個眼神就知道對方要傳球。
籌備
 為進行某件事而事先做好準備和計劃。（Prepare / Plan）
學生會正在努力籌備下星期的才藝表演，希望活動能順利進行。
委屈
 受到不公平的對待而心裏感到難過。（Feel wronged）
明明不是我打破花盆，卻被老師責備，讓我感到很委屈。
爭執
 各執己見，互相辯論不肯讓步。（Dispute / Argue）
我們為了決定專題研習的題目而發生了爭執，最後要請老師幫忙調解。
敷衍
 做事不認真，只求表面應付過去。（Perfunctory）
朋友認真向你請教問題時，你不應該隨便找個藉口敷衍他。
鼓勵
 激發並勉勵別人，使其增強信心。（Encourage）
在我測驗不及格感到失落時，好朋友不斷鼓勵我，讓我重新振作。
體諒
 設身處地為別人着想，並給予同情和理解。（Show understanding）
同學最近因為生病而缺交功課，我們應該多加體諒並主動借筆記給他。
📌 四字成語（不供詞填充用語）
成語 釋義 生活化例句
刮目相看
 指別人已有進步，不能再用老眼光去看他。
平時數學成績一般的明仔，這次竟然拿了滿分，真讓人對他刮目相看。
竊竊私語
 背地裏小聲地說話。
早會時，校長在台上講話，台下的同學卻在竊竊私語，這樣十分不尊重。
孤立無援
 形容單獨行事，得不到外力援助。
剛轉學到新學校時，我誰也不認識，偶爾會有一種孤立無援的感覺。
莫名其妙
 指事情很奇怪，使人不明白其中的道理。
他今天一言不發就退出了我們的WhatsApp群組，真是令人莫名其妙。
幸災樂禍
 指別人遭遇災禍，自己卻感到高興。
看到對手在比賽中失誤跌倒，我們不應該幸災樂禍，反而應該展現體育精神。
志同道合
 志向相同，思想一致。
參加動漫學會讓我認識了一班志同道合的好朋友，我們經常一起討論新番。
侃侃而談
 理直氣壯、從容不迫地說話。
雖然是第一次上台演講，但他毫不緊張，面對全校同學依然能侃侃而談。
面紅耳赤
 形容因急躁、害羞或發怒而臉紅的樣子。
為了哪個男團的舞步最齊整，她們兩個爭論得面紅耳赤。
悶悶不樂
 形容心情煩悶，不快樂。
因為不小心弄壞了朋友借給他的Switch遊戲機，他整天都悶悶不樂。
恍然大悟
 形容忽然明白過來。
聽了老師用打機來比喻這個數學概念後，我才恍然大悟，原來這麼簡單！

四、改寫句子（消滅英式中文邏輯）

以下句子帶有明顯的口語或「直譯英語」的痕跡。請將它們改寫為自然、流暢的中文書面語。

❌ 英式中文 / 口語入文
這件事讓我變得很生氣。
(This makes me very angry.)
✅ 自然書面語
這件事令我十分氣憤。
原句：我寫這封信是為了告訴你我不可以去你的party。
觀看答案
我寫這封信，是想告訴你我無法出席你的派對。
原句：明天的比賽我們會play our best，贏那個獎杯回來。
觀看答案
明天的比賽我們會全力以赴，把獎盃贏回來。
原句：我不覺得這是一個好主意，因為會用很多時間。
觀看答案
我認為這不是個好主意，因為十分耗時（/因為太浪費時間了）。
原句：他是一個很nice的人，總是help別人。
觀看答案
他為人友善，總是樂於助人。
原句：我的好朋友昨天沒有來學校，所以他miss了很重要的notes。
觀看答案
我的好朋友昨天缺席，因而錯過了十分重要的筆記。
原句：你可以給予我一些advice關於怎樣溫習嗎？
觀看答案
你可以給我一些關於溫習的建議嗎？
原句：我很sorry昨天忘了帶書給你。
觀看答案
昨天忘了把書帶來給你，我感到十分抱歉。
原句：那個老師講的話很make sense，大家都同意。
觀看答案
老師的話言之有理，大家都十分贊同。
原句：我們需要有一個meeting去傾這個project。
觀看答案
我們需要開會討論這個專題研習。
原句：我有很多功課要做，所以我感到很大壓力。
觀看答案
功課繁重令我感到壓力很大（/我因為功課繁重而感到壓力大）。

五、重組句子（建立書面語感）

請將以下散落的詞組，重新組合成語意通順、邏輯嚴謹的書面語句子，並加上適當的標點符號。

(1) 在操場上 / (2) 揮灑汗水 / (3) 籃球校隊的 / (4) 隊員們 / (5) 正在

(1) 溝通和包容 / (2) 才能 / (3) 建立 / (4) 互相的 / (5) 長久的友誼

(1) 令人 / (2) 校園生活 / (3) 豐富多彩的 / (4) 回味無窮 / (5) 總是
觀看答案
豐富多彩的校園生活總是令人回味無窮。
(1) 舉辦了 / (2) 學校 / (3) 為了減輕 / (4) 壓力 / (5) 同學們的 / (6) 一連串的活動
觀看答案
為了減輕同學們的壓力，學校舉辦了一連串的活動。
(1) 積極參與 / (2) 不但能 / (3) 課外活動 / (4) 擴闊社交圈子 / (5) 還能 / (6) 培養領導才能
觀看答案
積極參與課外活動不但能擴闊社交圈子，還能培養領導才能。
(1) 因為 / (2) 產生了 / (3) 我們 / (4) 一些小誤會 / (5) 缺乏溝通 / (6) 而
觀看答案
我們因為缺乏溝通而產生了一些小誤會。
(1) 主動 / (2) 當別人 / (3) 伸出援手 / (4) 我們應該 / (5) 遇到困難時
觀看答案
當別人遇到困難時，我們應該主動伸出援手。
(1) 充滿了 / (2) 歡聲笑語 / (3) 整個 / (4) 課室裡 / (5) 同學們的
觀看答案
整個課室裡充滿了同學們的歡聲笑語。
(1) 面對 / (2) 保持 / (3) 同儕間的 / (4) 競爭 / (5) 我們應該 / (6) 良好的心態
觀看答案
面對同儕間的競爭，我們應該保持良好的心態。
(1) 是 / (2) 學習生涯中 / (3) 同學們 / (4) 老師的 / (5) 鼓勵 / (6) 最大的動力
觀看答案
老師的鼓勵是同學們學習生涯中最大的動力。

六、段落理解（生活化閱讀）

【短文一】
上週五的班際籃球決賽，至今仍讓我記憶猶新。我們班面對實力強勁的對手，一開始便處於劣勢，比分大幅落後。中場休息時，大家垂頭喪氣，氣氛十分凝重。這時，隊長阿明站了出來，他沒有責備任何人，反而拍拍大家的肩膀，鼓勵我們：「比賽還沒結束，只要我們發揮平日排練的默契，一定能追回來！」這番話彷彿為大家注入了強心針。下半場，我們改變策略，加強防守，並透過精準的傳球創造得分機會。雖然最終我們還是以兩分之差落敗，但全班同學都為我們在場上展現的拼搏精神而歡呼。這次比賽讓我明白，輸贏固然重要，但團隊合作和永不言敗的精神才是最珍貴的收穫。
比賽一開始，作者的班級處於什麼狀況？
觀看答案
處於劣勢，比分大幅落後。
中場休息時，面對低落的士氣，隊長阿明做了什麼？
觀看答案
他沒有責備任何人，反而鼓勵大家只要發揮平日默契，一定能追回來。
下半場，作者的隊伍主要作出了哪些戰術改變？（寫出兩點）
觀看答案
改變策略加強防守；透過精準的傳球創造得分機會。
這場籃球比賽最終的結果是什麼？
觀看答案
最終以兩分之差落敗。
作者認為從這次比賽中學到最珍貴的東西是什麼？
觀看答案
團隊合作和永不言敗的精神。
【短文二】
新學期，我們班來了一位轉校生——傑克。他剛從英國回流香港，由於中文不太流利，總是獨自坐在角落，顯得格格不入。有一次小組專題研習，老師特意將傑克安排在我的組別。起初，討論過程十分吃力，他常常因為找不到合適的中文詞彙而急得面紅耳赤。為了幫助他，我們提議中英夾雜地交流，並慢慢教他一些常用的中文句式。漸漸地，傑克變得開朗起來，更在匯報時主動承擔了部分中文演講。看著他在台上自信的笑容，我深刻體會到，只要多一份體諒與耐心，語言絕不會成為建立友誼的障礙。
傑克剛來到班上時，為什麼會顯得「格格不入」？
觀看答案
因為他剛從英國回流，中文不太流利。
在小組專題研習時，老師作出了什麼特意的安排？
觀看答案
老師特意將傑克安排在作者的小組內。
起初討論時，傑克為什麼會急得「面紅耳赤」？
觀看答案
因為他找不到合適的中文詞彙來表達自己。
作者和組員用了什麼具體的方法來幫助傑克融入？
觀看答案
提議中英夾雜地交流，並慢慢教他一些常用的中文句式。
從傑克的轉變中，作者體會到了什麼與人相處的道理？
觀看答案
只要多一份體諒與耐心，語言絕不會成為建立友誼的障礙。
