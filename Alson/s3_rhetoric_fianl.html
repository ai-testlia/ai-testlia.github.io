第一章：「把」字句與「被」字句的高階轉換

概念重溫

在中文語法中，「把」字句和「被」字句是用來強調動作執行者（施事者）和動作承受者（受事者）之間的關係。到了小五階段，句子會加入很多形容詞和副詞來修飾，轉換時必須一字不漏地保留這些修飾語。

「把」字句：主動者 ＋ 把 ＋ 被動者 ＋ 動作及結果
「被」字句：被動者 ＋ 被 ＋ 主動者 ＋ 動作及結果
基礎例子（三年級程度）
弟弟把遙控車弄壞了。
遙控車被弟弟弄壞了。
高階例子（五年級程度：加入複雜修飾語）
火紅的太陽把一望無際的大地烤得火辣辣的。
一望無際的大地被火紅的太陽烤得火辣辣的。
⚠
常見陷阱
同學在轉換長句時，經常會遺漏修飾語（形容詞/副詞），或者把修飾語放錯位置。緊記：「誰的」和「怎樣的」必須跟著原本的名詞一起移動！
❌ 錯誤示範（遺漏修飾語）

原句：調皮的猴子把遊客手中那頂昂貴的帽子搶走了。
錯轉：帽子被猴子搶走了。

✅ 正確示範

原句：調皮的猴子把遊客手中那頂昂貴的帽子搶走了。
正轉：遊客手中那頂昂貴的帽子被調皮的猴子搶走了。

✏️ 挑戰區：高階句型轉換（10題）

指示：請根據括號內的指示，精準地改寫下列句子。注意修飾詞的位置。

1. 狂風暴雨把農田裡那些快要成熟的農作物全都摧毀了。（改為「被」字句）
2. 經驗豐富的消防員把困在火場三樓的小女孩安全地救了出來。（改為「被」字句）
3. 那隻飢腸轆轆的老虎把草原上那隻正低頭吃草的小綿羊撲倒了。（改為「被」字句）
4. 廚房裡那塊香噴噴的烤肉被貪吃的流浪狗偷偷地叼走了。（改為「把」字句）
5. 城市裡那些具有歷史價值的舊建築物被無情的推土機夷為平地。（改為「把」字句）
6. 班長昨天借給我的那本珍貴的科幻小說被我不小心弄破了幾頁。（改為「把」字句）
7. （重組句子成「把」字句） 識破了 / 機智的 / 騙局 / 警察 / 騙子的 / 把 / 輕而易舉地
8. （重組句子成「被」字句） 掩蓋了 / 白茫茫的 / 大地 / 大雪 / 被 / 整個 / 一夜之間
9. 找出錯處並改正：那輛在高速公路上飛馳的跑車把路旁的積水濺得路人一身都是。（改為被字句：路旁的積水被跑車濺得路人一身都是。）
10. 擴寫句子（先擴寫，再轉為「把」字句）：貓捉住了老鼠。（加入形容詞修飾貓和老鼠，並加入副詞修飾捉住）
查看答案
1. 農田裡那些快要成熟的農作物全都被狂風暴雨摧毀了。
2. 困在火場三樓的小女孩被經驗豐富的消防員安全地救了出來。
3. 草原上那隻正低頭吃草的小綿羊被那隻飢腸轆轆的老虎撲倒了。
4. 貪吃的流浪狗偷偷地把廚房裡那塊香噴噴的烤肉叼走了。
5. 無情的推土機把城市裡那些具有歷史價值的舊建築物夷為平地。
6. 我不小心把班長昨天借給我的那本珍貴的科幻小說弄破了幾頁。
7. 機智的警察輕而易舉地把騙子的騙局識破了。
8. 一夜之間，整個大地被白茫茫的大雪掩蓋了。
9. 錯處在於遺漏了修飾語。改正：路人一身都被那輛在高速公路上飛馳的跑車濺滿了路旁的積水。
10. 參考答案：身手敏捷的黑貓迅速地把那隻狡猾的老鼠捉住了。

第二章：關聯詞的精準運用（因果、轉折、承接）

三大複句關係剖析

複句類型 常用關聯詞 用法說明與示例
因果複句 因為(由於)...所以...
...原來...
既然...就...		 前句說明原因，後句說明結果。
例：由於哥哥平日刻苦訓練，所以在陸運會上奪得金牌。
轉折複句 雖然...但是(卻)...
儘管...還是...
...然而...		 後句的意思與前句相反或相對，出乎意料。
例：儘管前路充滿荊棘，他還是咬緊牙關勇往直前。
承接複句 先...然後...
一...就...
...便... / ...於是...		 按時間先後順序，連續發生的動作或事件。
例：大雨一停，孩子們就迫不及待地跑到遊樂場玩耍。
💡
考試技巧：辨析「儘管...還是...」與「既然...就...」
「儘管」表示退一步承認某種困難事實（轉折），結果依然不變；「既然」表示先提出一個已成定局的前提（因果推論），再引出應有的結論。
例：儘管下大雨，我們還是要上學。（轉折） / 既然下大雨，我們就取消野餐吧。（因果）
✏️ 挑戰區：關聯詞高階運用（10題）

指示：第一至五題選詞填空；第六至十題運用關聯詞把兩句合併為一句（可增刪標點或字詞）。

供詞：雖然...卻... / 由於...所以... / 先...然後... / 一...便... / 儘管...還是... / 既然...就...

1. ____________ 祖父年事已高，行動不便，但他 ____________ 堅持每天到公園散步鍛鍊身體。
2. ____________ 你已經答應了幫忙佈置會場，____________ 不應該臨陣退縮。
3. 媽媽回家後，____________ 把菜洗乾淨，____________ 才把肉放進鍋裡炒。
4. 那隻機警的小鳥 ____________ 聽見草叢裡傳來微小的聲響，____________ 立刻振翅高飛。
5. ____________ 海面颳起了猛烈的颱風，____________ 所有的渡輪服務都被迫暫停。
6. （合併句子 - 轉折）這道數學題看起來十分簡單。我花了半個小時也找不到正確的解題方法。
7. （合併句子 - 承接）賊人聽到遠處傳來警車的響號聲。賊人立刻丟下贓物拔腿就跑。
8. （合併句子 - 因果）溫室效應日益嚴重。全球兩極的冰川正以驚人的速度融化。
9. （合併句子 - 轉折）狐狸費盡唇舌用各種甜言蜜語哄騙烏鴉。烏鴉始終緊閉嘴巴不發一言。
10. 錯句改正：因為志明平時沒有溫習，還是這次考試成績一落千丈。
查看答案
1. 儘管 / 還是
2. 既然 / 就
3. 先 / 然後
4. 一 / 便
5. 由於 / 所以
6. 雖然這道數學題看起來十分簡單，但是我卻花了半個小時也找不到正確的解題方法。
7. 賊人一聽到遠處傳來警車的響號聲，便立刻丟下贓物拔腿就跑。
8. 由於溫室效應日益嚴重，所以全球兩極的冰川正以驚人的速度融化。
9. 儘管狐狸費盡唇舌用各種甜言蜜語哄騙烏鴉，烏鴉還是始終緊閉嘴巴不發一言。
10. 改正：因為志明平時沒有溫習，所以這次考試成績一落千丈。

第三章：敘事手法分析：倒敘法的魅力

甚麼是倒敘法？

倒敘法是指打破常規的時間順序，先在文章開頭交代事情的結局或最突出的片段，然後再按時間順序回溯事情的起因和經過。

➡️ 順敘法結構

事情的起因 ➔ 事情的經過 ➔ 事情的結局 ➔ 感受/道理

⏪ 倒敘法結構

事情的結局 ➔ 事情的起因 ➔ 事情的經過 ➔ 總結(感受/道理)

✅
寫作重點：為甚麼要用倒敘法？
1. 製造懸念：先告訴讀者一個出人意料的結局，引發讀者的好奇心，吸引他們一直讀下去找原因。
2. 突出主題：把最震撼、最感人的畫面放在開頭，給讀者留下深刻的印象。
✏️ 挑戰區：段落重組與分析（10題）

指示：第1至5題，請根據段落結構判斷其使用的是「順敘」還是「倒敘」；第6至10題為文章重組，請把它排列成一篇倒敘法的文章。

1. 昨天早上我睡過頭，匆忙出門時忘記帶乘車卡，結果遲到了被老師責備，這真是不好受的一天。 （順敘 / 倒敘）
2. 看著手背上一道深深的疤痕，我不禁回想起五年前那場驚險的單車意外…… （順敘 / 倒敘）
3. 台下響起了雷鳴般的掌聲，我手捧著金盃，淚水模糊了雙眼。記得三個月前，我還是一個連五線譜都不會看的初學者…… （順敘 / 倒敘）
4. 媽媽買了麵粉和雞蛋，我們一起搓麵團、入烤箱，最後烤出了香噴噴的曲奇餅。 （順敘 / 倒敘）
5. 爺爺總是珍而重之地擦拭著一個破舊的懷錶。原來，這個懷錶是他年輕時…… （順敘 / 倒敘）

6-10題：請把以下五個段落 (A-E) 重新排列，組成一篇通順且運用了倒敘法的文章。請寫出正確的字母順序。

A. 一個多月前，我在寵物店櫥窗看到牠，牠瘦弱的身軀和無助的眼神深深打動了我，於是央求媽媽讓我帶牠回家。
B. 經過這件事，我明白到照顧生命是一件嚴肅的責任，絕不能只有三分鐘熱度。
C. 自從有了「小黑」這隻活潑可愛的小狗陪伴，我的生活充滿了歡樂。但在這歡樂背後，其實隱藏著一段驚險的經歷。
D. 剛帶回家時，我十分勤奮地照顧牠。可是半個月後，我因為沉迷電子遊戲，忘記了給牠餵食和添水。
E. 有一天放學回家，我發現小黑奄奄一息地趴在地上。獸醫說牠嚴重脫水，差點沒命。看著牠虛弱的樣子，我流下了悔恨的眼淚。
排序： _____ ➔ _____ ➔ _____ ➔ _____ ➔ _____
查看答案
1. 順敘
2. 倒敘
3. 倒敘
4. 順敘
5. 倒敘
6-10. 排序為： C ➔ A ➔ D ➔ E ➔ B
（解析：C 是現在的結局/總起，A 是起因，D 和 E 是經過，B 是最後的總結感受。這構成了倒敘法的標準結構。）

第四章：修辭與人物描寫手法深究

1. 修辭手法：排比與反復

修辭手法 定義 示例
排比 三句或以上，結構相同或相似、語氣一致的句子並排在一起。能增強文章氣勢。 天上的雲，有的像奔騰的白馬，有的像溫馴的綿羊，有的像兇猛的獅子。
反復 重複使用某個詞語或句子，目的是突出重點，強調強烈的感情。 「成功了！成功了！我們的火箭終於升空了！」科學家們激動地喊道。

2. 人物描寫手法

要讓筆下的人物活靈活現，我們不能只寫「他很高興」或「他很生氣」，而應該透過具體的描寫來呈現。主要的描寫手法包括：

  • 語言描寫： 寫出人物說的話和說話時的語氣。
  • 動作描寫： 寫出人物具體的四肢或身體動作。
  • 表情(神態)描寫： 寫出人物五官的變化，如眉毛、眼睛、嘴角的活動。
綜合運用示例分析
得知測驗滿分後，小明高興得眉開眼笑，嘴角快要咧到耳根了（表情）。他興奮地一蹦三尺高，雙手用力地拍著（動作），大喊：「太棒了！我終於做到啦！」（語言）。
